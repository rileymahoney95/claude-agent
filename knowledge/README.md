# Claude Agent

Personal automation assistant for daily tasks and workflows.

## Directory Structure

- `.config/` - Configuration files (gitignored)
- `.data/` - Runtime data, context, logs (gitignored)
- `automations/` - Automation scripts and workflows
- `scripts/` - Utility and maintenance scripts
- `knowledge/` - Documentation and templates
- `links/` - Symlinks to external directories

## Quick Start

Access your files through symlinks:
- `links/desktop` → ~/Desktop
- `links/documents` → ~/Documents
- `links/projects` → ~/Projects

## Usage

Run automations from the `automations/` directory.
Context is maintained in `.data/context/`.
